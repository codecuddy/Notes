C.     Generalizations

                i.         User visits AngularJS app

              ii.         The view presents the apps data through the use of expressions, filters, and directives.

             iii.         User clicks an element in the view, if the element has a directive it runs the function

             iv.         The function updates the state of the data

              v.         The view automatically changes and displays the updated data.

1.     Note: The page doesnt need to reload at any point!
